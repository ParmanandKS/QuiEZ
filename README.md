# ✨ QuiEZ - Dynamic Quiz Engine

> A lightweight, open-source, JSON-powered quiz system built for simplicity, speed, and scalability.

---

## 🚀 Features

- ⚡ Fully static web app (no backend required)
- 🌐 Runs directly in browser (works offline too)
- 📦 JSON-based dynamic quiz system
- 🧠 Supports multiple quizzes in one platform
- 🗂 Smart quiz dashboard for selection
- 🔄 Fully dynamic question rendering engine
- 📊 Auto score calculation system
- 🏁 Instant result generation after submission
- 🧩 Modular architecture (easy to extend & customize)
- 🔗 Clean separation of data (JSON) and logic (JS)
- 🌍 Ready for GitHub Pages deployment
- 📱 Responsive and lightweight UI
- 🎯 Works for any exam, subject, or practice test
